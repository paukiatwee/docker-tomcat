Docker Tomcat
=============

Docker image with latest JDK 8 and Tomcat 8

## Environments

### Runtime
* Operating System - `centos:centos7`
* Java - Oracle JDK

### Directory
* Java - `/opt/java`
* Tomcat - `/opt/tomcat`

### Variables
All variables defined in `setup.sh`
* Java Installer URL - `JAVA_URL`
* Tomcat Installer URL - `TOMCAT_URL`
